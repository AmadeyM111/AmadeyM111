Building production-grade AI systems at the intersection of Machine Learning, Large Language Models and Software Engineering.

My primary focus is designing autonomous AI agents capable of long-term memory, planning, retrieval, tool use, structured reasoning and reliable execution in production environments. I enjoy transforming research ideas and prototypes into scalable, maintainable systems with strong observability, evaluation and engineering practices.

Currently working as a Data Scientist, I combine ML, NLP and systems engineering to build intelligent applications powered by LLMs. My work spans Retrieval-Augmented Generation (RAG), agent architectures, orchestration, vector search, model evaluation, production inference pipelines and AI infrastructure.

Outside of work, I develop \*\*Jormung\*\* — an autonomous AI agent built on top of the Ouroboros runtime. The project explores long-running agents, memory management, multi-model routing, tool orchestration, controlled self-evolution, observability and reliable execution across Web, Docker and Telegram environments.
